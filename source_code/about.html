The goal of this website is to be able to identify country flags quickly. By selecting or deselecting characteristics about the flag, you can whittle the list down and identify the flag you are looking for. The characteristics include colors, shapes, and a few patterns which I've included in the shapes category.

This application was created for a final project for CS 701 at Boston University. The code is open source and available on GitHub.

View Code

How to use this site

The filter buttons allow for three states, selected, deselected, or neither. If you are not sure if a flag does or does not have a characteristic, leave it as neither.
